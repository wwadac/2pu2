# Telegram Реферальный Бот

Бот с реферальной системой, начислением 12 рублей за друга и выводом средств от 600 руб.

## Возможности
- Реферальная система с бонусом 12 руб/реферал
- Проверка подписки на канал (вкл/выкл в админке)
- Личный кабинет с балансом и статистикой
- Система вывода средств (мин. 600 руб)
- Админ-панель с управлением

## Установка
1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Создайте файл `.env` и укажите токен бота и ID админа
4. Запустите: `python bot.py`

## Команды
- `/start` - запуск бота
- `/admin` - панель администратора
